Mike's Utility Library

Mike's utility library is a collection of useful Maven and Java tools.

Components and Usage

Build Tools

A POM file with a set of Maven profile configurations for reliable and repeatable Maven builds.

Note that this POM file is not part of the parent / child relationship of the project to keep it isolated from the Maven build(s). This allows it to being used as a configuration dependency without the risk of circular dependencies.

Contents:

  • Maven license plugin configuration / BSD license file
  • Checkstyle rule set configuration
  • Findbugs global exclude filter
  • PMD rule set configuration

Maven Dependency

<groupId>org.beiter.michael.util</groupId>
<artifactId>build-tools</artifactId>
<version>1.2</version>

Util Library Parent

The utility library parent POM controls the build of the utility library.

Contents:

  • Maven (build) module version control and configuration
    • Compiler plugin
    • License plugin
    • Dependency plugin
    • Source plugin
    • JXR plugin
    • Checkstyle plugin
    • Findbugs plugin
    • PMD plugin
    • Taglist plugin
    • Cobertura plugin
    • Javadoc plugin
    • Surefire plugin
    • Project info reports plugin
    • Site plugin
  • Dependency Management: All dependencies for this project have to be listed in a <DependencyManagement> section.
  • Organization, licensing, and developer information
  • Source control settings
  • Issue management settings
  • Distribution management settings
